# Клиентское приложение - Агентство новостроек

React приложение для агентства недвижимости, специализирующегося на продаже квартир в новостройках.

## Технологии

- React 18
- Vite
- Tailwind CSS
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

## Функциональность

- Каталог новостроек с карточками проектов
- Фильтрация по району, цене, количеству комнат и статусу
- Детальная страница проекта
- Адаптивный дизайн

## Деплой на Vercel

Проект настроен для деплоя на Vercel. Конфигурация находится в файле `vercel.json`.

### Переменные окружения

Для работы приложения необходимо настроить переменную окружения `API_URL` в настройках проекта на Vercel:

1. Перейдите в настройки проекта на Vercel
2. Откройте раздел "Environment Variables"
3. Добавьте переменную:
   - **Name**: `API_URL`
   - **Value**: `https://admin-doman-horizont.ru/api` (или ваш URL API)
   - **Environment**: Production, Preview, Development (по необходимости)

Также поддерживается переменная `VITE_API_URL` (будет использована, если `API_URL` не задана).

### Локальная разработка

Для локальной разработки создайте файл `.env` в корне проекта:

```env
API_URL=https://admin-doman-horizont.ru/api
```
